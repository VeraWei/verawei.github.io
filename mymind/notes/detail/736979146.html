其实这是一篇近期看过的记录片的总结，跟题目没多大关系，哈哈。

原来还有纪录片这么个宝藏没有开发，每一个基本上都是1.5 倍速，用了一个月的时间居然看了这么多，真是不列不知道。。c 问我有什么分享的note么，发现没有，自己也真是就看个乐呵，然后看完就忘。想想可以写写。影片简介不再重复，分为几个部分介绍说说感受吧。

1. 艺术捣蛋篇：

《徒手攀岩》

徒手攀岩 (2018)
8.8
2018 / 美国 / 纪录片 / 伊丽莎白·柴·瓦沙瑞莉 金国威 / 亚历克斯·霍诺德 汤米·考德威尔

作为一个非常害怕突然死亡的人来说，这个真的是吓死我，提着气看他一点点的爬上去，生怕呼吸重了都会影响他脚步的精准度，热爱有的时候真的是很疯狂的一件事，self-sabotage 被极端放大就叫艺术了。

《Minding the Gap》

滑板少年 (2018)
8.8
2018 / 美国 / 纪录片 / 刘冰 / 基尔·约翰逊 刘冰

《Losers》

败局启示录 (2019)
9.0
2019 / 美国 / 纪录片 运动 / Mickey Duzyj

以为是另一个运动类纪录片，看到最后这是一个伦理片。。关于原生家庭以及家庭暴力这种迫害性真的是写一箩筐都写不完。相处模式这种东西都是会传染的，依然清晰的记得儿时的家庭暴力，记得那次次被打的经历，后来问过我爸为什么喜欢发脾气，控制不住还打人，原因千篇一律吧，大男子主义自己不如意且小时候就是被这么对待着长大的。哪一个都不是我的原因。传播的力量是多么的可怕。二十岁就结婚生子真的不是一个好主意。当你离开那个环境，当你经济独立，当你第三者视角去看待整件事，当你人格建立起来后，就不会再怕了。大家都是一个鼻子两条腿的生物而已。

《天梯：蔡国强的艺术》，《脸庞，村庄 Visages villages》

天梯：蔡国强的艺术 (2016)
8.6
2016 / 中国大陆 美国 / 纪录片 / 凯文·麦克唐纳 / 蔡国强 吴红虹
脸庞，村庄 (2017)
9.1
2017 / 法国 / 纪录片 / 阿涅斯·瓦尔达 让·热内 / 阿涅斯·瓦尔达 让·热内

美这种东西可大可小哇，大了壮观，小了精致，真是各有各的道理（这是一句屁话）对艺术，除了欣赏，发出“还可以这么玩”的感叹外，说不出一句话。希望哪天词汇量丰富到可以不用“我x”来表达。。

2. mindset 篇：

《Inside Bill's Brain: Decoding Bill Gates》

走进比尔：解码比尔·盖茨 (2019)
9.2
2019 / 美国 / 纪录片 / 戴维斯·古根海姆 / 比尔·盖茨 梅琳达·盖茨

祖师爷爷作为世界首富，他运用自己的资源，拆解问题找寻事情的解决办法，希望建一个很好的世界，夫人的思维方式弥补住他的gap，合作的刚刚好，看着让人很是敬佩，话说啊，非洲真让人操心。之前马老板说钱对他来说就是一个数字，他喜欢解决问题，a字头公司着实做了很多公益的事，比较想看看马爸爸接下来能玩出些什么花样。

《The Mind, Explained》

头脑解密 第一季 (2019)
8.5
2019 / 美国 / 纪录片 / 艾玛·斯通 咏给·明就仁波切

看完上一个，就看到了连环推荐，头脑解密。记得听过一句细思极恐的话：大脑是人体的指挥官，这句话是大脑告诉你的。该片从 记忆、梦境、焦虑、正念、迷幻剂 五个方面解析大脑，挺有意思的，提到了记忆宫殿，潜意识等等。都说大脑是人体最大的性器官，这很弗洛伊德。除了看心、看看脑，也是了解自己的一部分。

3. 还有希望篇：

《他乡的童年》

他乡的童年 (2019)
8.9
2019 / 中国大陆 / 纪录片 / 周轶君 / 周轶君

近些年国内的纪录片质量跟国产动画一样节节攀升，日本的童年教育让我有些吃惊，之前去日本的时候，看到一个老师牵着一队的小朋友往前走，觉得这个国家给人的规矩感真是从小就建立起来的，可是影片又向我展示的完全不是那么回事，大概是不同的教育形式在共存吧。然后是印度、以色列、英国、芬兰。我可能对印度有偏见然后抵抗了解接触，改天好好想想这偏见是哪来的，现在先有着吧。最让我嫉妒的是芬兰的小孩子，相比于其他国家很前卫的教育资源（潜意识里觉得需要用金钱地位来换），这个国家不一样！艺术普及、不以边框塑造小朋友的人格，保持性格的多样性的前提就是没有成功这个概念。。。

不禁让人感叹，现在的孩子都这么幸福了么？父母大概率的选择为其争取资源，然后有的国家还不用争取，呵。。酸。。

《人生果实》

人生果实 (2017)
8.7
2017 / 日本 / 纪录片 / 伏原健之 / 津端修一 津端英子

耄耋之年，携手前行，影片温馨点太多了，最好的感情是什么呢，可能只有离世之后，后人说是什么，那就是什么了吧。

“他喜欢吃土豆，我不喜欢吃”；

“她是我最好的女朋友”；

“谢谢你”；

“一点一点，从自己能做到的开始，缓慢而坚定”;

这片子不光温馨，还励志真的是真的是太扎心了。。好想要当他们的孙女。。。
